## Running on Local Development

1. Clone Frontend Repository - **https://github.com/Alamnzr123/fe_TestApp**
2. Clone Backend Repository - **https://github.com/Alamnzr123/be_TestApp**
3. NPM install
4. Adding Database
5. Run using npm
```
  npm run dev
```
